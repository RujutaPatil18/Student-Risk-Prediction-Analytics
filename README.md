# Student Risk Prediction and Academic Performance Analytics Dashboard

## Project Overview

This project was developed during my Data Analytics internship at Paarsh Infotech. The project focuses on analyzing student academic performance data, predicting student risk levels using Machine Learning, and visualizing insights through an interactive Tableau dashboard.

The system helps identify academically at-risk students based on factors such as attendance, internal marks, assignment scores, study hours, and academic behavior patterns.

---

## Objectives

* Analyze student academic data
* Predict student risk levels
* Identify attendance and study behavior patterns
* Build an interactive Tableau dashboard
* Generate meaningful academic insights

---

## Key Insights

* Students with higher attendance generally perform better academically
* Most students belong to the MEDIUM risk category
* Study hours positively affect academic performance
* High-risk students are comparatively fewer
